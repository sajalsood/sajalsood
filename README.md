## About

I am a Staff Software Engineer focused on platforms where governed data, semantic intelligence, and product workflows meet.

My recent work centers on:

- Snowflake Native Apps with Snowpark Container Services, Python services, multi-service APIs, app roles, scoped privileges, and secure deployment models.
- Semantiqa, a semantic intelligence platform for governed semantic layers and natural-language analytics through LLM-powered agents.
- Distributed data and AI systems using Kubernetes, FastAPI, event-driven workflows, Snowflake tasks, streams, observability, and resilient APIs.
- Product engineering for complex platform capabilities, with practical UX, reusable frontend modules, and maintainable release patterns.
